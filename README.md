# Web Application Analytics & UX Insights Dashboard

## Overview
Analyzed 100 user session records to identify patterns in conversion and bounce rates, and to understand user behavior across devices.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Microsoft Excel
- Power BI

## Key Insights
- Conversion Rate: 27%
- Mobile Users: 42%
- Highest Session Duration: ~127 seconds (Product Page)

## Visualizations
- Histogram
- Scatter Plot
- Bar Chart
- Pie Chart
- Line Graph
